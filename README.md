# 墨尔本大学课程回放字幕翻译插件

将课程回放的字幕实时翻译成中文，以 CC 字幕形式叠加显示在视频画面上（已由悬浮窗更新）。

## 支持范围

- **支持**：Lecture Capture（Lecture Capture 进入的回放页面）
- **不支持**：LMS Files 里直接上传的 MP4 文件（无字幕轨道，需要事事语音识别api）

## 安装

1. 下载或克隆本仓库到本地
2. 打开 Chrome，地址栏输入 `chrome://extensions/`
3. 右上角开启**开发者模式**
4. 点击**加载已解压的扩展程序**，选择本仓库文件夹
5. 插件安装完成，浏览器右上角会出现插件图标

## 配置百度翻译 API

插件使用百度翻译 API，每位用户需自行申请免费额度（每月 100 万字符免费）。

1. 前往 [百度翻译开放平台](https://fanyi-api.baidu.com/) 注册并创建应用
2. 获取 **APP ID** 和 **密钥（Secret Key）**
3. 点击浏览器右上角插件图标，填入 APP ID 和密钥
4. 点击**保存设置**

配置只需做一次，保存在本地浏览器中，不会上传到任何服务器。

## 使用方法

1. 在 LMS 进入任意课程，点击左侧 **Lecture Capture**
2. 打开任意一个回放视频
3. **确保 Echo360 播放器的字幕（CC）已开启**——点击播放器底部的 CC 按钮启用字幕，插件依赖字幕文本来源
4. 播放视频，中文翻译字幕会自动出现在视频画面底部，位于英文字幕上方

### 拖动字幕位置

鼠标移到中文字幕上，光标变为移动箭头，按住拖动即可调整位置。松手后位置自动保存，下次打开会恢复。

### 开启 / 关闭翻译

点击浏览器右上角插件图标，切换**启用字幕翻译**开关。关闭后刷新页面生效。

### 全屏

直接点击播放器全屏按钮即可，中文字幕会跟随进入全屏。

## 注意事项

- 首次打开回放页面时，插件会等待字幕容器加载，最多重试约 40 秒
- 翻译结果缓存在内存中，快进/回放已翻译过的段落会立即显示，无需重复调用 API
- 源语言自动检测，支持英语、泰卢固语（被迫支持）等任意语言翻译成中文
- 百度翻译免费版有每秒 1 次的 QPS 限制，连续快速跳转时偶尔会有短暂延迟
